# gabrielsvieira01.github.io

Site pessoal hospedado no GitHub Pages, com algumas ferramentas feitas pra apoiar a faculdade de Medicina.

## O que tem aqui

- **Agenda interativa do 8º período** (`agenda/`): calendário semanal com filtros por categoria e grupo, link compartilhável, modo foto e modo de incorporação (embed).
- **Simulados de OSCE do 7º período** (`osce7p/`): sem relação com a agenda.
- **Índice de navegação** (`index.html`): lista os arquivos `.html` do repositório.
